### With the <span style="color:#F1C40F;">**Resizer**</span> **hold down** the <build/interact> button and **release** when done.

![build showcase](https://github.com/Paperweightt/quick_resize_mcbe/blob/main/assets/build_preview.gif)

### **Shift** to <span style="color:#BA372A;">Delete</span> walls

![delete showcase](https://github.com/Paperweightt/quick_resize_mcbe/blob/main/assets/delete_preview.gif)

### Detailed Instructions

1. Select a single block
2. Pick a side of the block, this will control how the block will stretch
3. Hold down the build/interact button
4. Move where you are looking to where you want the wall to end
5. If you want to delete, hold shift
6. Release the build/interact button
